The command line :
is a text-based interface used to interact with a computer's operating system. It allows you to enter commands and execute them ,to access the command line, you can use the built-in terminal application on your computer. On Windows, you can open the Command Prompt or PowerShell
Basic Navigation: the directory system is a hierarchical tree-like structure that starts from the root directory ("/") and branches out into subdirectories. To navigate through this directory system, you can use basic commands in the terminal the common command : cd,ls,pwd,rm,mkdir,rmdir
More About Files :
Linux is an Extensionless System and Case Sensitive The extension of a file usually consists of 2 to 4 characters that appear after a period at the end of the file name, indicating the type of file it is. Here are some frequently used file extensions: file.exe - an executable program file.txt - a plain text file file.png, file.gif, file.jpg - an image file. file obtain information about what type of file a file or directory is.
Manual Pages :
The set of pages known as manual pages provide a detailed explanation of every command available on your system, including their functions, execution methods, and accepted command line arguments. Although some of the manual pages can be challenging to comprehend, they are generally uniform in their content. man Look up the manual page for a particular command.man -k Do a keyword search for all manual pages containing the given search term ./ Within a manual page, perform a search for 'term'nAfter performing a search within a manual page, select the next found item.
File Manipulation :
To create a file in Linux, you can use the "touch" command followed by the filename. For example, to create a file called "example.txt", you would type "touch example.txt" in the terminal. To remove a file, you can use the "rm" command followed by the filename. For example, to remove the "example.txt" file, you would type "rm example.txt". To rename a file, you can use the "mv" command followed by the current filename and the new filename. For example, to rename the "example.txt" file to "newfile.txt", you would type "mv example.txt newfile.txt". To copy a file, you can use the "cp" command followed by the source filename and the destination filename. For example, to copy the "example.txt" file to a directory called "mydir", you would type "cp example.txt mydir". To move a file, you can use the "mv" command followed by the source filename and the destination directory. For example, to move the "example.txt" file to a directory called "mydir", you would type "mv example.txt mydir".
